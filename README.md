# 🦷 Clínica Sorriso Pleno

Site institucional moderno e responsivo para uma clínica odontológica de excelência.

🔗 **[Acessar Demo ao Vivo](https://clinica-demo-three.vercel.app/)**

## ✨ Funcionalidades

* Menu fixo com transição suave no scroll
* Hero animado com link rápido para WhatsApp
* Cards de serviços interativos
* Contadores de dados animados no viewport
* Carrossel automático de depoimentos
* Formulário de agendamento com validação e simulação de envio
* Integração nativa com iframe do Google Maps
* Modo Escuro/Claro (Dark Mode) com preferência salva via `localStorage`
* Conformidade LGPD: Banner de consentimento de cookies e modal de Política de Privacidade
* Layout totalmente responsivo para Mobile, Tablet e Desktop
* Botão flutuante animado para atendimento rápido no WhatsApp

## 🛠️ Tecnologias

* **HTML5**
* **CSS3**: Layouts flex/grid, animações nativas, variáveis CSS (`:root`)
* **JavaScript**: Vanilla ES6+, sem dependências externas
* **APIs Nativas**: Intersection Observer API para detecção de rolagem
* **Fontes**: Google Fonts (Inter)
* **Mapas**: Google Maps Embed

## 🚀 Como rodar localmente

Clone o repositório em sua máquina local:

```bash
git clone https://github.com/patrick-witt/clinica-demo.git
```

Abra a pasta do projeto e dê um duplo clique no arquivo `index.html` para executá-lo diretamente em seu navegador padrão. Nenhuma instalação de dependência (NPM/Yarn) é necessária.
